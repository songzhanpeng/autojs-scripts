## 使用说明

1. **安装 AutoJS**
   - 确保你的手机已经安装 AutoJS 应用。

2. **导入脚本**
   - 将需要使用的脚本下载到本地。

3. **运行脚本**
   - 打开 AutoJS 应用。
   - 导入已下载的脚本。
   - 运行脚本，根据需要进行配置。

4. **注意事项**
   - 在使用脚本之前，请确保了解脚本的具体功能和效果。
   - 部分脚本可能需要在特定应用或场景下使用，请仔细阅读相关说明。
